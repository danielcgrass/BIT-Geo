##Mileage Tracker

This project is designed to track the mileage a Semi Truck travelled per state. 

The main ‘track_points’ table has the following fields :
————————————————————————
id		:  integer id for Semi Truck Lat/Lon input
lat		:  latitude in decimal degrees
lon		:  longitude in decimal degrees
speed 	:  integer int to store speed of truck
time		:  datetime type to store time and date of each value
